SRA — Skill Runtime Advisor 是一个运行在 AI Agent 旁的守护进程，在 Agent 收到用户消息时自动分析语义、推荐最匹配的 Hermes Skill，并通过 pre_tool_call 钩子系统在校验工具调用前检查技能遵循度。
