## 📋 Sprint Backlog

### Story 1: Tool 层 SRA 校验 (SRA-003-01) — 🔴 P0 — 3天

**目标**: Agent 在调用 write_file/patch/terminal/execute_code 前自动校验技能遵循度

**任务分解**:
1. SRA Daemon 新增 `POST /validate` 端点
   - 接收 `{tool, args, loaded_skills[], task_context}`
   - 返回 `{compliant, missing[], severity}`
2. 创建 `plugins/sra-guard/plugin.py` Hermes 插件
3. 集成到 Hermes `pre_tool_call` 钩子系统
4. 非阻塞降级：SRA 不可用时不影响工具执行
5. 编写测试用例

**文件变更**:
- 新增: `skill_advisor/runtime/endpoints/validate.py`
- 新增: `plugins/sra-guard/plugin.py`
- 修改: `skill_advisor/runtime/daemon.py`
- 修改: `tests/`

### Story 2: 文件类型技能映射 (SRA-003-02) — 🔴 P0 — 2天

**目标**: 通过文件扩展名自动推导所需技能

**任务分解**:
1. 创建 `FILE_SKILL_MAP` 映射表 (`config/skill_map.json`)
2. 集成到 `POST /validate` 校验中
3. 支持用户自定义扩展映射
4. 编写测试用例

**文件变更**:
- 新增: `skill_advisor/skill_map.py`
- 新增: `skill_advisor/config/skill_map.json`

### Story 3: 技能使用轨迹记录 (SRA-003-03) — 🟡 P1 — 2天

**目标**: 追踪技能加载和实际使用情况，检测模式偏差

**任务分解**:
1. 增强 `POST /record` 端点支持 `action: "viewed"|"used"|"skipped"`
2. Hermes `skill_view()` 拦截自动上报
3. Hermes 工具调用后自动上报
4. 场景记忆记录使用序列

**文件变更**:
- 修改: `skill_advisor/runtime/daemon.py`
- 修改: `skill_advisor/memory.py`
- 修改: Hermes `run_agent.py`

### Story 4: 长任务上下文保护 (SRA-003-04) — 🟡 P1 — 2天

**目标**: 长任务执行中定期重注入技能建议，防止上下文漂移

**任务分解**:
1. Hermes `run_conversation()` 每 5 轮自动重查询 SRA
2. SRA 新增 `recheck()` 基于对话摘要
3. `[SRA 提醒]` 轻量级注入
4. 可配置提醒间隔

**文件变更**:
- 修改: Hermes `run_agent.py`
- 修改: `skill_advisor/advisor.py`

### Story 5: SRA 契约机制 (SRA-003-05) — 🟡 P1 — 2天

**目标**: 任务开始时生成技能契约，明确推荐技能优先级

**任务分解**:
1. `POST /recommend` 返回新增 `contract` 字段
2. `build_contract()` 方法生成契约
3. 契约信息格式化到 `rag_context`
4. 契约在 `/validate` 校验时参考

**文件变更**:
- 修改: `skill_advisor/runtime/daemon.py`
- 修改: `skill_advisor/advisor.py`

### Story 6: 运行时力度体系 (SRA-003-06) — 🟡 P1 — 3天

**目标**: 4 维度可配置的运行时力度体系，支持 5 种预设一键切换模式

**任务分解**:
1. 设计力度配置 schema（`config/schema.py`）
2. 实现力度引擎（`runtime/force.py`）— 4 维度：scope/action/contract/confidence
3. 定义 5 种预设模式：seedling/casual/normal/strict/fortress
4. 集成到 `/validate` 端点：力度影响返回的 severity 和 action
5. 集成到 `/recommend` 端点：力度影响 confidence 门槛过滤
6. 新增 `sra config set` CLI 命令
7. 编写测试用例

**文件变更**:
- 新增: `skill_advisor/config/schema.py`
- 新增: `skill_advisor/runtime/force.py`
- 修改: `skill_advisor/runtime/daemon.py`
- 修改: `skill_advisor/cli.py`
- 新增: `tests/test_force.py`
