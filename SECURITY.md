# 安全策略

## 支持的版本

| 版本 | 支持状态 |
|------|----------|
| v1.1.x | ✅ 积极维护 |
| v1.0.x | ✅ 安全更新 |
| < v1.0 | ❌ 不再支持 |

## 报告漏洞

如果发现安全漏洞，**请不要公开提交 Issue**。请通过以下方式报告：

1. 发送邮件至项目维护者（见 GitHub Profile）
2. 或使用 GitHub 的 [Private vulnerability reporting](https://docs.github.com/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability) 功能

### 报告内容

- 漏洞类型和严重程度
- 复现步骤
- 受影响的版本
- 可能的修复方案（可选）

我们会在 48 小时内回复，并尽快修复。
